What it includes

Fixtures, match reports, team analysis, player analysis, standings, Wyscout views, and transition animations.

Goztepe Hub

Pre-match scouting, rival scout, post-match reporting, and trend tracking for Goztepe workflows.

Analytics engine

Custom xG inference, possession chains, xT, PPDA, field tilt, style classification, and visual reports.

Run locally

GitHub Pages is static, so it cannot run the Dash server. Clone the repository and start the apps locally: 

python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
python göztepehub/app.py
